# Loan-Management-System
A secure, role-based Loan Management System built using ASP.NET Core Web API and Angular that enables online loan application, approval workflows, EMI calculation, repayment tracking, and real-time loan status monitoring with robust authentication and authorization.

Features:
Online loan application and status tracking
Role-based access (Admin, Loan Officer, Customer)
Loan approval and rejection with remarks
Automated EMI calculation and repayment tracking
Secure authentication and authorization

Tech Stack:
Frontend: Angular
Backend: ASP.NET Core Web API
Database: SQL Server
Security: JWT Authentication

Improved transparency and accuracy
Secure and centralized data management
